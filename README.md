# kartavyasonar.github.io

Personal portfolio website for **Kartavya Sonar** — Full Stack Developer & AI Engineer.

**Live:** https://kartavyasonar.github.io

## Stack
- Pure HTML / CSS / JavaScript — no build step, no dependencies
- Canvas API for animated coding setup hero
- Deployed via GitHub Pages

## Features
- Animated canvas coding desk hero (60fps)
- Project detail overlays on click
- Terminal animation in contact section
- Scroll-reveal animations throughout
- Fully responsive

## Deploy
Just push `index.html` — GitHub Pages serves it instantly.
